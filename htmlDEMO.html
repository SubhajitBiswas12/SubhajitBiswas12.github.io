HTML Tags Reference with Interactive Examples

Text Formatting Tags

  • <b> — Bold text, no attributes.
  • <strong> — Strong importance, no attributes.
  • <i> — Italic text, no attributes.
  • <em> — Emphasized text, no attributes.
  • <u> — Underlined text, no attributes.
  • <mark> — Marked/highlighted text, no attributes.
  • <sub> — Subscript text, no attributes.
  • <sup> — Superscript text, no attributes.
  • <ins> — Inserted text, no attributes.
  • <del> — Deleted text, no attributes.
  • <br> — Line break, no closing tag or attributes.
  • <hr> — Horizontal rule.
    • size: height of the line (e.g. size="3")
    • width: width in px or % (e.g. width="50%")
    • color: color of the line (e.g. color="red")

Document Structure Tags

  • <!DOCTYPE> — Declares the HTML version (e.g., <!DOCTYPE html>).
  • <html> — Root element.
    • lang: language code (e.g., lang="en")
  • <head> — Contains meta, title, styles, scripts.
  • <title> — Title shown in browser tab.
  • <link> — External resources.
    • rel: relationship (e.g., "stylesheet")
    • href: URL to the file
  • <style> — Internal CSS.
  • <script> — JavaScript (inline or src).
  • <body> — Page content.
    • bgcolor: background color
    • text: text color
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Page</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body bgcolor="lightyellow" text="black">
    Hello World!
  </body>
</html>

List Tags

  • <ul> — Unordered list.
    • type: disc | circle | square
  • <ol> — Ordered list.
    • type: 1 | A | a | I | i
      start: starting number (e.g., start="5")
  • <li> — List item.
    • value: custom number (only in <ol>)

Link and Media Tags

  • <a> — Anchor (link).
    • href: URL
    • target: _blank | _self | _parent | _top
    • title: tooltip text
  • <img> — Image.
    • src: image URL
    • alt: alt text
    • width, height: dimensions
  • <marquee> — Scrolling text (deprecated).
    • direction: left | right | up | down
    • behavior: scroll | slide | alternate

Table Tags

  • <table> — Table container.
    • border: size in px
    • width: table width
    • cellpadding: spacing inside cells
  • <caption> — Table title.
    • align: top | bottom (deprecated)
  • <thead>, <tbody>, <tfoot> — Table sections.
  • <tr> — Table row.
    • align: left | center | right
      bgcolor: color
  • <td> — Table cell.
    • colspan, rowspan
